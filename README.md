# CapAhead Site

Public marketing website for CapAhead, operated by HAC Labs.

This repository is separate from the CapAhead app repository so the website and app can be deployed independently.

## Play Console submission note

When submitting CapAhead to Google Play, keep the Data Safety form aligned with the Privacy Policy:

- Finance entries are device-first and HAC Labs does not keep a server copy of local finance entries.
- Google Play handles purchases, subscriptions, trials, billing, cancellation, and payment processing.
- GitHub Pages and Cloudflare may process basic technical logs to serve and protect the website.
